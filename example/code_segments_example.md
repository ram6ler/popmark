# code_segments_example.md

## Segment 1

Here is some embedded Dart code...

```dart
for (var i = 1; i <= 3; i++) {
  print('$i. Hello, world!');
}
```

```text
1. Hello, world!
2. Hello, world!
3. Hello, world!
```
